# Tiempos Arranz — Panel de Entrenamientos

Dashboard web que muestra los datos del Google Sheet en tiempo real (auto-refresco cada 60 s).

## Despliegue

Esta web está pensada para desplegarse en Vercel, Netlify o similar. No requiere configuración adicional.

## Desarrollo local (opcional)

```bash
npm install
npm run dev
```

## Build de producción

```bash
npm run build
```

Los archivos generados quedan en `dist/`.
